# JoguinhoCheffs

## Visão Geral

JoguinhoCheffs é um jogo com temática culinária.

## Estrutura do Projeto

```
/
├── CLAUDE.md
└── (arquivos do projeto)
```

## Comandos Essenciais

```bash
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev

# Build de produção
npm run build

# Rodar testes
npm test
```

## Diretrizes de Desenvolvimento

- Escreva código limpo e legível
- Siga os padrões de estilo existentes no projeto
- Adicione testes para novas funcionalidades
- Faça commits pequenos e descritivos

## Branch de Trabalho

- Branch principal de desenvolvimento: `main`
- Novas features: `feature/<nome-da-feature>`
- Correções: `fix/<descricao-do-bug>`
